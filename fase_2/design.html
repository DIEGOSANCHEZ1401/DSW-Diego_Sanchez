Fundamentos de CSS,

¿Qué son las hojas de estilo en cascada C CSS significa hojas de estilo en cascada (cascading style sheets),
y es muy importante entender la palabra cascada. La forma en que se comporta la cascada es la clave para
comprender el CSS. En algún momento trabajarás en un proyecto y encontrarás que el CSS que pensabas
que debería aplicarse a un elemento no funciona.

Frameworks CSS

Que es Frameworks CSS Es un conjunto de componentes para interfaz de usuario que te permite desarrollar aplicaciones
basadas en HTML de manera rápida y efectiva. Todos los componentes son compatibles en móviles y diversos navegadores web
