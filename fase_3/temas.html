¿ QUE ES FLEX BOX ?



Es un método diseño de página unidimensional para compaginar elementos br en filas o columnas los elementos de contenido se ensanchan para rellenar br el espacio adicional se cogen para caber Durante mucho tiempo, las únicas br herramientas disponibles para crear diseños CSS y posicionar elementos con buena br compatibilidad entre browsers eran float y position. Sin embargo, estas herramientasbr tienen algunas limitaciones muy frustrantes, especialmente cuando se trata de responsividad. br Algunas tareas que consideramos básicas en un diseño, como centrar verticalmente un elemento br secundario en relación con un elemento principal o hacer que los elementos secundarios ocupen br la misma cantidad de espacio, o que las columnas tengan el mismo tamaño independientemente de br la cantidad de contenido interno
